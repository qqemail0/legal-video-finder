# Security Policy

## Supported Versions

The current `main` branch receives security fixes.

## Reporting

Open a private security advisory on GitHub or contact the maintainer through the repository issue tracker if private advisory access is unavailable.

## Scope

Please report:

- Arbitrary code execution.
- Unsafe file writes.
- Secret exposure.
- Injection vulnerabilities in provider parsing.
- Unsafe URL handling.

Do not report requests to add piracy sources as security issues; those are out of scope and will be rejected.
